# AlertyAI 🤖✅

AlertyAI is a premium, AI-powered task management assistant built exclusively for Android. It combines local efficiency with cloud intelligence, allowing you to manage your life using voice, images, and smart chat. 

<Note>
AlertyAI is exclusively available as a mobile application for Android. There is currently no web version available.
</Note>

## Why AlertyAI?

Traditional task managers require you to manually type out every detail, set dates, and organize lists. AlertyAI flips this paradigm by bringing **AI directly to your task management**.

Whether you want to speak your tasks aloud, snap a picture of a handwritten schedule, or just chat with an AI assistant to organize your day, AlertyAI handles the heavy lifting so you can focus on getting things done.

- **Fast & Intuitive:** Designed natively for Android using Modern Jetpack Compose.
- **Privacy First:** Hybrid storage keeps your local tasks on-device in a secure Room database.
- **Smart Parsing:** Natural language processing understands dates, times, and context automatically.
